Note:
	1. Secara default, program ini tidak mempunyai nama database. Oleh karena itu, pada saat pertama kali mengakses program ini,
	   Anda akan di redirect ke halaman pembuatan database. Anda hanya perlu memasukkan :
			- Hostname. 		Default: localhost. 			Contoh: localhost, dll
			- Username. 		Default: root. 					Contoh: sa, root, dll
			- Password. 		Default: '' -> empty string. 	Contoh: sa, root, dll
			- Nama database. 	Isikan sesuai dengan keinginan
	
	2. Silahkan login dengan menggunakan user admin.
			- Nomor pegawai : admin
			- Password : admin
			
	3. Supaya fitur cetak dapat terpakai, diharapkan untuk mematikan pop-up blocker browser.
	4. Selamat mencoba :)
